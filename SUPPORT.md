# Support

## Where To Ask Questions

Use GitHub issues for repository-specific bugs, regressions, and feature requests.

## What This Repository Supports

This repository supports the baseline itself, including:

- Dev Container configuration
- setup and update scripts under `.devcontainer/`
- repository documentation
- shell test coverage and CI behavior

## What Is Usually Out Of Scope

- support for custom changes made only in downstream forks
- vendor account, billing, or sign-in problems for external AI services
- host machine issues unrelated to repository behavior
- general Docker or VS Code support questions with no repository-specific reproduction

## Before Opening An Issue

Please include:

- host OS
- VS Code version if relevant
- whether the failure happens during on-create, post-create, post-start, or CI
- the command you ran
- the relevant log output
