# Code of Conduct

## Purpose

This repository is maintained as a professional, respectful, and technically focused collaboration space.
Everyone participating in issues, pull requests, reviews, and discussions is expected to help keep it that way.

## Expected Behavior

- Be respectful in technical disagreement.
- Critique ideas, code, and assumptions rather than people.
- Keep feedback specific, actionable, and relevant to the repository.
- Assume good intent unless there is clear evidence otherwise.
- Share context when reporting bugs, regressions, or breaking behavior.

## Unacceptable Behavior

- Harassment, intimidation, or personal attacks
- Discriminatory, hateful, or demeaning language
- Repeated off-topic disruption
- Publishing private information without permission
- Abusive behavior toward maintainers, contributors, or users

## Scope

This code of conduct applies to repository spaces, including issues, pull requests, review comments, and other project communication channels managed through GitHub.

## Enforcement

Maintainers may remove comments, close threads, or block participation when behavior violates this code of conduct.

If you need to report a conduct issue, contact the maintainers through the repository owners on GitHub and include links or screenshots where possible.
