# 完全自律評価 — 設計書

## 目的
従来の「親主導逐次実行」評価では、親が順序制御を担保していたため、
「SKILL.md を読んだ subagent が自立的にネスト制約を理解し行動できるか」を測定できていない。

この評価では SKILL.md 全文を subagent の context に渡し、
subagent が SKILL.md の記述だけから正しい行動（順序・状態隔離・曖昧さ処理）を
自律的に判断・実行できるかを検証する。

## 従来評価との違い

| 項目 | 従来（親主導） | 完全自律 |
|------|--------------|---------|
| 順序制御 | 親が各subagentを順次起動 | subagent自身が判断 |
| spec/quality review | 親が別subagent起動 | subagent内で自己完結またはスキップ判断 |
| ネスト制約対応 | 親が制御 | subagentがSKILL.md記述から理解・対応 |
| 評価対象 | 手順遵守率 | SKILL.mdの指示伝達力 |

## 成功基準
- [critical] 全要件の自律的な認識・実行
- Accuracy ≥ 83%（iter-1ベースライン相当以上）
- 重大な手順逸脱なし（順序逆転、ファイル上書きなど）

## 3シナリオ
- A: 独立タスク3つ（順序制御の自律性）
- B: 同一ファイル連続タスク（状態隔離の自律性）
- C: 曖昧な設定値（代替行動の自律性）
