# サポート

## 質問去哪里

リポジトリ固有のバグ、回帰、機能リクエストには GitHub イシューを使用してください。

## このリポジトリがサポートする範囲

このリポジトリは、ベースライン自体をサポート対象とします：

- Dev Container 設定
- `.devcontainer/` 以下のセットアップおよびアップデートスクリプト
- リポジトリドキュメント
- シェルテストカバレッジと CI 動作

## 通常範囲外のもの

- 下流フォークでのみ行われたカスタム変更のサポート
- 外部 AI サービスのベンダーアカウント、請求、サインイン問題
- リポジトリ固有の再現がないホストマシンの問題
- リポジトリ固有の再現がない一般的な Docker または VS Code サポート質問

## イシューを開く前に

以下を含んでください：

- ホスト OS
- 関連する場合は VS Code のバージョン
- 失敗が on-create、post-create、post-start、または CI のいずれで発生するか
- 実行したコマンド
- 関連するログ出力
