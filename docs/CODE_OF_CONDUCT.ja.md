# 行動規範

## 目的

このリポジトリは、プロフェッショナルで尊重のある、技術的に焦点を当てた協調の場として維持されています。
イシュー、プルリクエスト、レビュー、議論に参加するすべての人々が、その状態を保つために協力することが期待されます。

## 期待される行動

- 技術的な不同意においては尊重ある態度で臨む。
- 人ではなくアイデア、コード、仮定を批判する。
- フィードバックは具体的かつ実行可能で、リポジトリに関連するものに保つ。
- 明確な証拠がない限り、良い意图を前提とする。
- バグ、回帰、断続的な動作を報告する際は文脈を共有する。

## 容認できない行動

- ハラスメント、威嚇、または個人攻撃
- 差別的、憎悪的、または貶めるような言葉遣い
- 繰り返し行われるトピック外の問題行為
- 許可のないプライベート情報の公開
- メンテナー、コントリビューター、またはユーザーへの虐待的行為

## 適用範囲

この行動規範は、イシュー、プルリクエスト、レビュコメント、GitHub で管理されているその他のプロジェクトコミュニケーション渠道を含む、リポジトリの空間に適用されます。

## 執行

メンテナーは、この行動規範に違反する行動に対して、コメントの削除、スレッドのクローズ、参加のブロックを行う場合があります。

行動規範の問題を報告する必要がある場合は、GitHub のリポジトリ所有者に連絡し、可能な限りリンクまたはスクリーンショットを添えてください。
