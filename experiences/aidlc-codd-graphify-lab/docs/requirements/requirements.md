TaskFlow Mini — Requirements

- Users can create tasks with title, description, status, and due date.
- Users can mark tasks as todo, doing, or done.
- Users can list tasks by status.
- The system exposes a small command-line interface.
- The implementation language is Python.
- The project must keep design documents and source code traceable.
